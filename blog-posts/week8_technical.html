How computers work

08/17/2014

What's a computer?

A computer is an object containing a microprocessor that accepts inputs and computes operations to produce outputs.

What makes up a computer?

Most computers consist of a:

  • Case -- holds all of a computer's parts
  • Motherboard -- is the primary circuit board inside your a computer. All components connect through the motherboard.
  • Power supply
  • Central Processing Unit (CPU) -- contains the microprocessor
  • Random Access Memory (RAM) -- the CPU uses to store things as it computes opereations
  • Drive -- used to store data that's not in use
  • Cooling system -- cools the CPU
  • Cables -- connect all the components

Operating System

The operations system (OS) is that you use to control the computer after it boots. The OS is responsible for:

  • Memory management -- coordinates the flow of data in and out of RAM.
  • Device management -- provides a software-based interface between the computer's internal components and each device connected to the computer like a keyboard or mouse.
  • Storage management -- directs where data should be stored permanently on hard drives.
  • Application interface -- provides data exchange between software programs that meet it's requirements and the PC. An application must be programmed to work with the application interface for the operating system you're using. This is why you see this in an application's requirements "Windows Vista or later," or "only works on 64-bit operating systems."
  • User interface (UI) - provides a way for you to interact with the computer.
